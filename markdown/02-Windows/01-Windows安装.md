# Windows安装

## 使用u盘

1. 在[next.itellyou.cn](https://next.itellyou.cn/)下载镜像（迅雷）
2. 通过[rufus](https://rufus.ie/zh)刻录到u盘

## 使用微PE

1. 在[next.itellyou.cn](https://next.itellyou.cn/)下载镜像（迅雷）
2. 通过[微PE](https://www.wepe.com.cn/)直接安装

## 安装注意事项

1. 一定要先备份数据
2. 建议先拆掉其他硬盘
3. 选择旧版安装
4. 删除C盘前面的分区（包括C盘）
5. 显示未分配空间即可安装
